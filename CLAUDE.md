## Commands

```bash

npm run build    # Build production site to ./dist/
npm run preview  # Preview production build locally
```

## Architecture

This is an **Astro** project using TypeScript (strict mode), Node >= 22.12.0.

- `src/pages/` — File-based routing; each `.astro` or `.md` file becomes a URL route
- `src/components/` — Reusable Astro/UI components (create as needed)
- `public/` — Static assets served at root (favicons, images, etc.)
- `.astro/` — Generated by Astro; do not edit
- `dist/` — Production build output; do not commit

Astro renders pages to static HTML at build time by default. The project is in early development with only a single index page.
